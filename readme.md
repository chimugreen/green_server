## 🎯 프로젝트 주제

> **소셜 네트워크 기반 TodoList**  
> → 함께 목표를 공유하고, 서로를 응원하며 성장할 수 있는 협업형 일정 관리 서비스

---

## 💡 프로젝트 목적

현대인은 혼자서 계획을 세우고 꾸준히 실천하기 어려운 경우가 많습니다.  
본 프로젝트는 **소셜 네트워크 기능을 결합한 TodoList 시스템**을 통해, 사용자가 **서로의 할 일을 공유하고 응원하며 지속적인 동기부여를 받을 수 있는 환경**을 조성하는 것을 목표로 합니다.

---

## 📄 프로젝트 개요

- SNS 기반의 TodoList 웹 애플리케이션
- 친구의 할 일을 확인하고 좋아요/댓글/응원을 통해 상호작용
- 혼자가 아닌 ‘함께 계획을 실천하는 환경’ 제공
- 개인의 동기부여 + 사회적 유대감 강화

---

## 🔑 주요 기능

- ✅ 회원가입 / 로그인 (JWT 인증)
- ✅ Todo 생성 / 수정 / 삭제
- ✅ 공개 / 비공개 설정
- ✅ 친구 추가 및 팔로우 기능
- ✅ 댓글 / 좋아요 / 응원 기능
- ✅ 진행률 시각화 대시보드
- ✅ 반응형 웹 디자인 (PC/모바일 대응)

- UX 중심 개발: 사용자 몰입도 높은 UI/UX 설계
